# Accident-Hotspot-Analysis

## Overview

Accident Hotspot Analysis is a data-driven project that identifies high-risk accident-prone locations using traffic accident data from Andhra Pradesh and Telangana. The project applies data preprocessing, geospatial analysis, and machine learning clustering techniques to detect accident hotspots and visualize them on an interactive map.

## Features

* Traffic accident data preprocessing and cleaning.
* Geospatial analysis using latitude and longitude coordinates.
* Identification of accident-prone zones using clustering algorithms.
* Interactive hotspot visualization on maps.
* Data-driven insights to support traffic safety and urban planning.

## Methodology

1. Collected and analyzed traffic accident datasets.
2. Performed data cleaning and preprocessing.
3. Extracted geographical coordinates (latitude and longitude).
4. Applied clustering techniques to identify accident hotspots.
5. Visualized accident-prone areas using interactive maps.

## Project Workflow

Data Collection → Data Preprocessing → Geospatial Analysis → Clustering → Hotspot Detection → Interactive Visualization
